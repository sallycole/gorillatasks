XML Upload Instructions

Each curriculum includes basic details (name, description) and an ordered sequence of tasks for students to complete.

Format your curriculum in an XML file as shown in the example below. Save with a .xml extension before uploading.

  • Grade levels shown are all available options - include only those that apply
  • Task descriptions are optional and can be left empty
  • Task URLs are optional but recommended for online content
  • Action tags must contain one of: Read, Watch, Listen, or Do
  • Tasks should be ordered from first to last in the sequence
